# Notes 

A notes creator running in console. 

The project use Go Lang and sqlite3. 

## Build 

To create the executable run:

```console
go build 
```


## Usage:

### Create a note 

Create a note using the parameter "-a".

```console
notes -a "my new note"
```

### Search note 

Search a note by a text "-s".

```console
notes -s "note"
```
